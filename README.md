# Data Analysis Project - Personal Expense Tracker

## Objective
Analyze personal expenses and visualize spending patterns.

## Tools Used
- Python
- Pandas
- Matplotlib

## Features
- Data cleaning
- Data analysis
- Visualization (Bar, Pie, Line charts)
- Insight generation

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run:
   python main.py
