# Personal Expense Analysis Report

## Overview
This project analyzes personal expenses to understand spending patterns.

## Key Insights
- Highest spending category: Bills
- Food expenses are frequent but moderate
- Transport cost is consistent
- Entertainment spending is occasional but high

## Visualizations
- Bar chart shows category comparison
- Pie chart shows expense distribution
- Line chart shows daily trends

## Conclusion
Better budgeting is needed for shopping and entertainment.
